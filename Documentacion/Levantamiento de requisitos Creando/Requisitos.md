La empresa creando desea desarrollar una aplicación web para promocionarse, así como promocionar, ofrecer y administrar los servicios que ofrecen.

# Requisitos funcionales
### 1. La aplicación debe tener una barra superior en la navegación
En esta barra deben haber enlaces para acceder a todos las paginas de la aplicación
### 2. La aplicación debe tener una [[Pagina Principal|pagina principal]] en la que aparezca:
- Una breve descripción de la empresa
- Una lista con los servicios que se ofrecen y enlaces para ver los detalles de cada uno.
- Enlaces a otros sitios web de la pagina que le puedan interesar a los clientes
### 3. La aplicación debe tener paginas para [[Oferta de servicios|detallar los servicios ofrecidos]]
Debe haber una de estas paginas por cada uno de los servicios principales ofrecidos
En esta pagina aparecen explicaciones y descripciones sobre ellos, incluyendo las actividades que se realizan, como benefician a la empresa y el talento humano necesario para llevarlas a cabo
### 4. La aplicación debe tener la posibilidad de [[Administracion de servicios|administrar y supervisar los servicios]] contratados
Cada cliente que tenga contratados los servicios de la empresa debe poder acceder a supervisar las actividades necesarias par llevarlas a cabo, así como el progreso de su realización

### 5. La aplicación debe tener una [[Bolsa de trabajadores]] en la que ofrecer vacantes a trabajos disponibles

### 6. La aplicación debe tener un sistema de suscripción
Debe ser capaz de enviar correos a los usuarios que lo deseen para notificar en caso de una nueva vacante en la empresa, un nuevo blog o un mensaje importante relación a un servicio contratado

# Requisitos no funcionales
- La aplicación debe tener un tono serio, **moderno** y empresarial para inspirar confianza a los clientes
- Debe ser ligera, para que sea fácilmente usable desde dispositivos con poco ancho de banda o mala conexión a la red
- Debe ser segura, de forma que cada usuario solo pueda acceder leer o modificar la información que le corresponde
